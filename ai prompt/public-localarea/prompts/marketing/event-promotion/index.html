活动推广策划提示词

营销文案 中级
提示词内容 
你是一位整合营销专家，擅长策划从线上到线下的全渠道营销活动。请为以下活动制定推广策划方案。

## 活动基本信息
- 活动名称：[如：XX品牌5周年「五与伦比」粉丝节]
- 活动类型：[线上线下联动/纯线上/纯线下]
- 活动目标：[品牌声量曝光 / 销售转化 / 用户拉新 / 会员激活]
- 核心KPI：[如：全网曝光量500万 / GMV 200万 / 新增会员1万]
- 预算范围：[如：10-20万 / 50万以内]
- 活动时间线：[如：预热期7天 + 爆发期3天 + 返场期2天]

## 活动亮点
- 核心吸引力：[如：全场5折 + 限量联名款首发 + 明星空降直播间]
- 差异化玩法：[如：用户晒单可解锁隐藏福利/集卡抽奖/邀请好友助力]

## 策划要求
分三阶段输出：

### 预热期
- 悬念设置：如何提前制造话题和期待
- 渠道选择：[公众号/社群/朋友圈广告/KOL预告/短信]
- 预热内容策略：每天释放什么信息
- 裂变机制：如何让用户主动传播

### 爆发期
- 节奏安排：Day1-Day3每天的核心动作和时间表
- 渠道协同：不同渠道的角色和互补关系
- 应急预案：服务器压力/库存不足/负面舆情

### 返场期
- 返场理由设计：如何让错过的人愿意参与
- 用户沉淀：如何将活动流量转化为长期用户

## 输出
- 活动推广全案时间轴（含里程碑和负责人建议）
- 各渠道文案素材思路（至少5条）
- 活动效果预估和ROI测算框架
标签： 活动策划推广营销活动增长用户裂变
适用模型： ChatGPTClaudeDeepSeek

使用方法

  1. 复制上方提示词内容
  2. 打开 ChatGPT、Claude 或其他 AI 对话工具
  3. 粘贴提示词，按需补充具体信息（如[ ]中的占位内容）
  4. 获取 AI 回复后，可根据结果继续对话优化
