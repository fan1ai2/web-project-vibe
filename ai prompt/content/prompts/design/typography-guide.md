你是一位排版设计专家，精通中英文排版和屏幕阅读体验优化。请为以下项目提供排版设计方案。

## 项目信息
- 项目类型：[如：新闻资讯App / 企业官网 / 数据后台 / 电子书阅读器]
- 主要内容类型：[长文阅读为主 / 数据表格为主 / 图文混排 / 短内容流]
- 平台与屏幕：[Web (1440px主屏) + 移动端 / 仅移动端 / 响应式全平台]
- 品牌调性：[如：现代极简 / 传统典雅 / 科技未来 / 温暖手作]

## 排版设计要求
1. **字体选择**
   - 标题字体（Heading Font）：推荐2-3个备选方案，说明每种字体的性格和适用性
   - 正文字体（Body Font）：推荐2-3个方案，注明可读性评分
   - 中英文字体配对建议（如中英文混排场景）
   - 等宽字体（Monospace）：用于代码/数据展示

2. **字体层级系统（Type Scale）**
   - 使用Major Third (1.25) 或 Perfect Fourth (1.333) 比例定义字号层级
   - h1-h6、body、caption、overline的具体字号/行高/字重
   - 移动端字号调整策略

3. **排版规则**
   - 正文行宽建议（每行多少个汉字/字符）
   - 行高（line-height）设置
   - 段落间距与首行处理
   - 标题与正文的间距关系

## 输出格式
- 字体推荐表（字体名称、性格描述、适用场景、免费/商用授权说明）
- Type Scale表格（层级/字号/行高/字重/用途）
- CSS排版基础变量（可直接复制使用）
