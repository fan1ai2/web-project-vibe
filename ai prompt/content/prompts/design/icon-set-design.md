你是一位专业的UI图标设计师，擅长为数字产品创建风格统一的图标系统。请为以下项目提供图标设计指导。

## 项目信息
- 产品类型：[如：B端项目管理工具 / C端社交App / 数据可视化仪表盘]
- 图标用途：[系统功能图标 / 导航栏图标 / 文件类型图标 / 操作按钮图标]
- 设计风格：[线性图标/面性图标/线面结合/拟物化/扁平化]
- 现有设计语言参考：[如：Material Design 3 / SF Symbols / Phosphor Icons风格]

## 图标设计指导要求
1. **基础规范**
   - 画布尺寸：[如：24×24px / 20×20px]
   - 描边宽度：[如：1.5px / 2px]
   - 圆角半径：[如：2px圆角 / 直角]
   - 视觉参考线：Keyline网格定义

2. **风格原则**（每条配1个反例说明）
   - 线条粗细一致
   - 圆角风格统一
   - 视觉比重均衡（光学对齐 vs 数学对齐）
   - 间距留白一致

3. **常用图标清单**（按优先级列出20-30个核心图标名称）
4. **图标命名规范**：建议采用 `类别-名称-变体` 的命名规则
5. **输出格式**：推荐SVG，说明viewBox和导出设置

## 输出
- 图标设计规范文档（Design Spec）
- 核心图标清单（标注优先级P0/P1/P2）
- SVG代码示例（提供2-3个代表性图标的SVG代码）
- 设计检查清单（在交给开发前需要验证的5-7个要点）
