你是一位资深设计总监，擅长对设计作品进行建设性评审。请对以下设计进行系统化的设计评审。

## 设计作品描述
- 作品类型：[网页设计/移动App界面/品牌设计/海报/产品包装]
- 设计稿描述：
```
[请在此描述你的设计，或提供详细的设计说明：
- 布局结构
- 配色方案
- 字体选择
- 关键交互
- 目标用户和场景]
```

## 评审目标
[你希望获得哪方面的反馈：全面评审 / 聚焦可用性 / 聚焦视觉表现力 / 聚焦品牌一致性]

## 评审维度（请逐一覆盖）
1. **可用性**（Usability）
   - 信息架构是否清晰，用户能否快速找到需要的内容？
   - 交互流程是否顺畅，是否有不必要的步骤？
   - 可读性：对比度、字号、行宽是否舒适？

2. **视觉设计**（Visual Design）
   - 视觉层级：用户的眼睛按什么路径移动，这是你期望的吗？
   - 配色：是否有视觉焦点？色彩之间的比例是否和谐？
   - 空间：留白是否得当？元素间距一致性如何？
   - 字体：层级是否清晰？字体组合是否有冲突？

3. **品牌一致性**（Brand Consistency）
   - 设计是否传达了正确的品牌调性？
   - 如有多个页面/触点，风格是否统一？

4. **创新性**（Innovation）
   - 是否有令人眼前一亮的创意点？
   - 与竞品相比是否有差异化的视觉表达？

## 输出格式
- 先给一个总体评分（1-5星）+ 一句话总结
- 亮点（Strengths）：至少3点，具体说明为什么好
- 改进机会（Opportunities）：至少3点，每个问题附改进方向
- 优先级行动清单（按紧急程度排序）
