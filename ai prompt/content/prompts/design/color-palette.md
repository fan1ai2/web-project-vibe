你是一位色彩设计专家，精通色彩心理学和配色调和理论。请为我生成一套配色方案。

## 配色需求
- 项目/品牌名称：[如：CloudDesk云桌面管理平台]
- 行业领域：[如：科技/SaaS / 医疗健康 / 教育培训 / 餐饮美食 / 时尚美妆]
- 品牌个性关键词：[如：专业、可靠、高效、科技感 / 温暖、自然、手工、有机]
- 配色偏好：[如：以蓝色为主色调 / 希望用绿色体现环保理念 / 需要高对比度的深色主题]
- 使用场景：[Web应用界面 / 移动App / 品牌VI印刷 / 数据可视化图表]

## 配色方案要求
1. 主色（Primary）：1个，占设计约60%的使用比例
2. 辅助色（Secondary）：1-2个，占约30%
3. 强调色（Accent）：1个，用于CTA按钮等关键元素，占约10%
4. 中性色：浅/深背景各一套（至少各5个层级）
5. 语义色：成功绿、警告橙、错误红、信息蓝各1个
6. 所有颜色提供HEX值和HSL值
7. 检查WCAG AA对比度合规性（正文不低于4.5:1，大文本不低于3:1）

## 输出格式
- 色板总览（所有颜色以色块网格形式排列，标注HEX/HSL）
- 主要颜色搭配示例（文字色+背景色组合，标注对比度比值）
- 色彩使用规范（什么颜色用在什么地方）
- 明暗主题对比预览（如需要）
