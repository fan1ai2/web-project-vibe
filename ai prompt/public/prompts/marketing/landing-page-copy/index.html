落地页文案提示词

营销文案 中级
提示词内容 
你是一位CRO（转化率优化）专家兼文案写手，擅长撰写高转化率的落地页文案。请为我撰写一套完整的落地页文案。

## 产品/服务信息
- 产品名称：[如：FlowAI - 智能工作流自动化平台]
- 一句话价值主张：[如：用AI连接你所有的办公工具，自动完成重复工作]
- 目标用户：[如：中小企业运营负责人，每天被跨平台数据搬运困扰]
- 核心痛点（Top 3）：
  1. [如：每天花2小时在不同工具间手动搬运数据]
  2. [痛点2]
  3. [痛点3]
- 核心功能/解决方案（Top 3）：
  1. [如：一键连接200+办公工具，无需代码]
  2. [功能2]
  3. [功能3]
- 社会证明：[如：已服务3000+企业 / 客户包括XX和XX / 好评率98%]
- 定价信号：[如：免费版可用 / 专业版199元/月起 / 提供免费试用]

## 文案结构要求
按照经典落地页Swipe File结构撰写：

1. **Hero区**：主标题（Headline）+ 副标题（Subheadline）+ CTA按钮文案 + 首屏视觉建议
2. **问题加剧区**：2-3句话放大痛点，让用户感到"这就是在说我"
3. **解决方案展示区**：产品作为"英雄"登场，3个核心功能+对应好处
4. **社会证明区**：用户logo墙/好评引言/数据证明的文案
5. **风险消除区**：打消顾虑（免费试用/退款保证/数据安全承诺）
6. **最终CTA区**：强化行动动机，降低行动门槛

## 输出
- 每个区块的完整文案
- A/B测试优先级建议（建议先测Hero区的哪2个版本）
- 微文案建议（表单提示文字/按钮hover状态文案/成功页感谢文案）

使用方法

  1. 复制上方提示词内容
  2. 打开 ChatGPT、Claude 或其他 AI 对话工具
  3. 粘贴提示词，按需补充具体信息（如[ ]中的占位内容）
  4. 获取 AI 回复后，可根据结果继续对话优化
