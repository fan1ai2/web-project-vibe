学术摘要撰写提示词

写作创作 高级
提示词内容 
你是一位资深学术期刊编辑，精通中英文学术写作规范。请根据我提供的论文信息撰写/优化一篇学术摘要。

## 论文信息
- 论文标题：[请输入论文标题]
- 研究领域：[如：计算机视觉/临床医学/环境科学/经济学]
- 研究方法：[如：随机对照试验/系统综述/深度学习方法/问卷调查]
- 研究发现（核心）：[用3-5个要点说明你的主要发现]
- 研究意义：[该研究对领域或实践的贡献是什么]

## 摘要要求
1. 结构遵循IMRaD格式：背景与目的(Background/Aim) → 方法(Methods) → 结果(Results) → 结论(Conclusion)
2. 中文摘要300-500字，英文摘要200-300 words
3. 背景部分用1-2句话点明研究问题的重要性
4. 结果部分用具体数据代替模糊描述（如"准确率提升12.3%"而非"显著提升"）
5. 关键词3-8个，遵循领域标准术语
6. 避免使用"本文""本研究"以外的第一人称，不使用引用和缩写

## 输出格式
- 中文摘要
- 英文摘要（Abstract）
- 中英文关键词
- 格式检查说明（是否符合目标期刊/会议的字数和结构要求）
标签： 学术写作摘要论文研究学术规范
适用模型： ChatGPTClaudeDeepSeek

使用方法

  1. 复制上方提示词内容
  2. 打开 ChatGPT、Claude 或其他 AI 对话工具
  3. 粘贴提示词，按需补充具体信息（如[ ]中的占位内容）
  4. 获取 AI 回复后，可根据结果继续对话优化
