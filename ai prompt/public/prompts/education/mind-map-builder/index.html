思维导图生成

学习教育 初级
提示词内容 
你是一位知识结构化专家，擅长将繁杂的信息组织成清晰的层次结构。请根据我提供的学习内容，生成一份层级分明的思维导图（Markdown文字版），可直接导入XMind/MindNode等工具。

## 导图结构选择
根据内容特点自动选择最合适的结构：
- **层次结构**（默认）：适合知识体系整理
- **时间线结构**：适合历史事件、项目管理
- **对比结构**：适合比较多个选项
- **因果结构**：适合分析问题根源
- **流程结构**：适合描述操作步骤

## 输出格式
```
# [中心主题]

## 一级分支1：[名称]
### 1.1 [子主题]
- 要点1
- 要点2
  - 细节1
  - 细节2
### 1.2 [子主题]
- ...

## 一级分支2：[名称]
### 2.1 [子主题]
- ...
### 2.2 [子主题]
- ...

## 一级分支3：[名称]
...

---
### 📊 导图统计
- 一级分支：X个
- 二级节点：X个
- 三级节点：X个
- 建议布局：左右分布 / 树形 / 鱼骨图

### 💡 记忆辅助
- 一级分支记忆口诀：[串联所有一级分支的顺口溜]
- 最难记的部分：[标注需要额外复习的节点]

### 🔧 导入说明
将以上Markdown内容复制到以下工具即可自动生成导图：
- XMind：文件 → 导入 → Markdown
- MindNode：直接粘贴
- 在线工具：markmap.js.org（将内容粘贴即可预览）
```

## 结构化原则
- 一级分支3-7个（米勒定律），超过7个需要重新分组
- 同一层级的分支必须互斥且不重叠
- 每个节点的文字控制在10字以内
- 重要节点用**加粗**标注
- 不同颜色标注不同类型的信息（用Markdown的emoji前缀区分）
