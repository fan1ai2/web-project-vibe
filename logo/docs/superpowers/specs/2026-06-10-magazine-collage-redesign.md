# 实验拼贴杂志风 UI 重设计

## 目标
将首页和创作台从"空洞的极简"升级为"独立设计杂志 + 实验拼贴 + 网格破坏"风格。

## 视觉语言
- 汉字作为核心图形，字号极致对比（1.2rem ~ 6rem）
- 元素越界、倾斜、错位，故意打破网格
- 双层边框模拟套色不准印刷效果
- 朱砂红 `#c7513b` 斜切色块点缀

## 首页

### Hero
- 标题三行拆解，大小极端对比
- 整体偏右 10%，左侧竖排英文微文字
- 巨大淡色几何圆被裁切一半

### Showcase
- 不均匀网格：2 大 + 3 标准 + 3 小
- 卡片随机微旋转，hover 回正
- 显示品牌首字、名称、风格标签

### 三步流程
- Z 字形错位排列
- 背景巨字水印 01/02/03
- 图标溢出到相邻区域

### CTA
- 朱砂红色块右下斜切
- 按钮在色块外，hover 滑入

## 创作台

### 筛选面板
- details 展开向左溢出 24px
- 选中 chip 印章效果

### 预览画布
- rotate(-1deg) + 双层投影
- 双层边框模拟套色不准
- 生成按钮越界 20px

### 历史面板
- 条目 cascading 倾斜排列
- 不规则圆角缩略图

## 动效
- 加载 stagger spring 弹入
- 滚动反直觉缩放
- 标题 hover 字符跳动
- 按钮 hover 色块斜切
- 生成 shake 动画
- reduced-motion 全部降级

## 约束
- 保留现有 Hugo 结构、数据、JS 逻辑
- 只改 CSS 和模板 HTML
- 不引入新依赖
