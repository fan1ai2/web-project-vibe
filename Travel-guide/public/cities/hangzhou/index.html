杭州旅游攻略2026 | 必去景点、美食推荐与行程规划

2026年最新杭州旅游攻略，专为自由行游客精心整理的完整指南。系统涵盖西湖十景等核心景点、灵隐寺门票价格与交通路线、龙井村九溪十八涧徒步攻略与西溪湿地游览建议，推荐楼外楼西湖醋鱼和知味观东坡肉等正宗杭州美食，附3日精华游及5日深度游行程规划。

01城市概览

杭州，浙江省省会，中国七大古都之一，被马可·波罗称为"世界上最美丽华贵之天城"。建城史超过2200年，五代吴越国和南宋都曾在此建都。西湖与京杭大运河两大世界文化遗产共存于同一座城市，这在全国绝无仅有。

作为江南文化的核心代表，杭州的气质是含蓄而精致的——西湖不张扬却处处是画，龙井茶清淡而回味悠长，杭帮菜讲究原味不靠重口调味。2026年杭州地铁已开通12条线路，覆盖萧山机场、火车东站、西湖景区和西溪湿地，自由行比五年前方便了不止一个档次。老城区巷子里的小吃店和西湖边的茶山步道，才是这座城市最值得花时间的地方。

02必去景点推荐

01

西湖风景名胜区

西湖是杭州的灵魂。环湖一周约10公里，景区由一山（孤山）、二塔（雷峰塔、保俶塔）、三堤（白堤、苏堤、杨公堤）、十景构成。推荐路线：从龙翔桥地铁站出来步行3分钟到湖滨，沿湖顺时针走——先到断桥残雪上白堤，穿过孤山（西泠印社、浙江省博物馆都在此），接着走苏堤春晓（2.8公里，六座桥起伏有致），到花港观鱼看红鲤，最后在雷峰塔（门票40元，登塔俯瞰西湖全景）结束。环湖也可以骑共享单车，边走边停约2-3小时骑完一圈。游船上三潭印月岛（船票+登岛55元）是看1元人民币背面图案的最佳方式。

免费（小景点另收费） · 地铁1号线龙翔桥站 / 凤起路站 · 1天
02

灵隐寺 + 飞来峰

灵隐寺是杭州香火最旺的寺庙，东晋咸和元年（公元326年）创建，距今近1700年。注意门票分两段：先进飞来峰景区（45元，含永福寺、韬光寺和飞来峰石窟造像），进灵隐寺需另购30元香花券。大雄宝殿内的释迦牟尼木雕坐像高24.8米，是中国最大的香樟木雕坐像。寺庙早上7:00开门，建议赶早到达，8:30后旅行团大量涌入体验会差很多。出来后可就近吃一碗素面。

飞来峰45元 + 灵隐寺30元 · 公交7路 / 407路直达 · 半天
03

龙井村 + 九溪十八涧

龙井村是西湖龙井茶的核心产区，坐公交27路或87路到龙井茶室站下车即可。村中茶农会热情招呼你进门喝茶，一般免费品尝，不买也没有关系。从龙井村沿九溪十八涧步道往下走是经典徒步线，全程约5公里缓坡下行不费力，沿途是茶园、溪流和枫树。秋天（10月底到11月）枫叶红了最美，春天茶叶嫩绿也很养眼。终点是九溪烟树，然后走到九溪公交站坐车回市区。建议穿运动鞋，避开雨天。

免费 · 公交27路 / 87路至龙井茶室站 · 半天
04

西溪国家湿地公园

西溪湿地距西湖不到5公里，是中国第一个国家湿地公园，水道纵横、芦苇丛生、白鹭成群。推荐坐摇橹船（100元/小时/船，可坐4-6人）穿行水道，船工会沿途讲解。秋季（10-11月）是西溪最美时段——芦苇花全白了，配着柿子树挂满红柿子，就是传说中的"秋芦飞雪"。夏季蚊虫比较多，务必带驱蚊水。整个景区走下来约3-4小时。

80元（船票另购60元） · 地铁3号线西溪湿地南站 · 半天
05

宋城景区

宋城是一个以南宋文化为主题的大型仿古景区，核心卖点是《宋城千古情》大型实景演出——被称作"世界三大名秀"之一。建议下午2-3点入园，先逛市井街看街头表演，傍晚吃饭，晚上看19:00或20:30场的演出（提前30分钟入场抢中间位置）。宋城适合对演艺和热闹氛围感兴趣的游客，如果冲着历史文化去的可能会失望——这里本质上是主题乐园而不是文博景点。

320元（含《宋城千古情》演出） · 公交4路 / 308路至宋城站 · 半天（下午+晚上）

♠西湖十景

西湖十景始于南宋，每一景都有一个诗意的名字。春夏秋冬、晨昏雨雪，不同时节、不同时辰，西湖有十种不一样的美。

苏堤春晓

春天苏堤上桃花盛开，2.8公里漫步或骑行，湖光山色尽收眼底

断桥残雪

冬日雪后桥面半融半白，远看像桥断了。白娘子与许仙相遇的地方

曲院风荷

夏日荷花盛开，清风送来荷香与酒香。北山路沿线最佳赏荷处

平湖秋月

白堤西端，中秋之夜在平台上看月亮从湖面升起，三面环水视野开阔

雷峰夕照

傍晚雷峰塔在夕阳下镀上金边，对面净慈寺钟声回荡，西湖最美日落机位

十景之外还有：柳浪闻莺、花港观鱼、南屏晚钟、双峰插云、三潭印月。环湖一圈约15公里，骑车2小时或步行4-5小时。

03美食推荐

西湖醋鱼— 人均约150元

杭帮菜的头牌。草鱼先在清水中饿养两天去土腥味，再用沸水汆熟浇上糖醋汁。楼外楼1848年开店，周总理曾在此宴请外宾，草鱼78元/条，鳜鱼版按季节约150-200元。口味偏甜酸，北方人第一次吃可能会觉得寡淡——这不是你的问题，是这道菜本就不走重口路线。预算有限的话，外婆家（人均60元）的西湖醋鱼也不错，口味更大众化。

楼外楼（孤山路30号）

东坡肉— 人均约80元

苏东坡在杭州做官时留下的菜谱。五花肉切成方块，加黄酒、酱油、冰糖慢炖两小时以上，成品色泽红亮、肥而不腻、酥烂到筷子一夹就断开。知味观是百年老字号（1913年创立），东坡肉18元/客，配一碗白米饭是绝配。新丰小吃（解放路店）的东坡肉更实惠，10元/客。不建议在景区内吃东坡肉，价格翻倍且品质不稳定。

知味观（仁和路83号）

片儿川— 人均约25元

杭州人的日常面食，雪菜+笋片+瘦肉片做浇头，汤清面劲。菊英面馆上过《舌尖上的中国》，一碗18-25元。面的灵魂在雪菜——必须是倒笃菜（杭州特有的腌雪菜），配冬笋片才是最正宗的版本。菊英早上6点开门，上午10点前到不用排长队。慧娟面馆（望江路店）的片儿川也很能打，汤底更浓郁，15元/碗。吃的时候加一勺本地辣酱，风味立刻上来了。

菊英面馆（中河南路12号）

龙井虾仁— 人均约80元

用清明前后的龙井新茶与新鲜河虾仁同炒，虾仁白玉般晶莹，带着一丝茶香。绿茶餐厅的龙井虾仁68-88元/份，餐厅本身坐落在茶园里，窗外就是大片茶山，环境比菜色更值得体验。杭州酒家（延安路店）的龙井虾仁78元/份，虾仁更大颗。新白鹿餐厅只要38元/份，味道也不错。这道菜适合搭配一壶龙井茶，边喝边吃，比单吃更有杭州味道。

绿茶餐厅（龙井路83号）

04交通出行

机场交通

杭州萧山国际机场距市中心约27公里，地铁1号线直达（约50分钟到龙翔桥站，票价8元），机场大巴20元到武林门（约40分钟），打车约100-120元。高铁主要停靠杭州东站（地铁1号线/4号线/6号线交汇）和杭州站/城站（地铁1号线/5号线）。

市内地铁

2026年杭州地铁运营12条线路，覆盖主要景点和交通枢纽。手机下载"杭州地铁"App或直接用支付宝/微信乘车码刷码进出站，每次2-8元。公交市区线路票价2元，支持乘车码支付。杭州的公共自行车（小红车）第1小时免费，第2小时1元，西湖边有密集的租车点。

出行贴士

景区交通：西湖景区周末和法定节假日实行单双号限行（外地车牌），自驾务必提前查清楚限行规则。电瓶车环湖一圈40元/人，招手即停。打车起步价13元/3公里，市区内一般20元以内搞定。

05行程推荐

3日精华游

第1天

Day 1：西湖环湖一日

龙翔桥地铁站 → 断桥残雪 → 白堤（1公里）→ 孤山（西泠印社、浙江省博物馆，免费）→ 楼外楼午餐 → 苏堤（2.8公里）→ 花港观鱼 → 雷峰塔（40元，登塔看日落）→ 晚上湖滨银泰晚餐 + 西湖音乐喷泉

第2天

Day 2：灵隐禅寺 + 龙井茶园 + 九溪徒步

公交7路到灵隐寺（早上7:30前到）→ 飞来峰石窟 → 灵隐寺 → 永福寺 → 知竹素面馆午餐 → 公交27路到龙井茶室站 → 龙井村喝茶 → 九溪十八涧徒步（5公里，1.5-2小时）→ 河坊街晚餐

第3天

Day 3：西溪湿地 + 河坊街 + 南宋御街

地铁3号线到西溪湿地南站 → 东区（3-4小时，坐摇橹船）→ 河坊街小吃 → 南宋御街民国建筑群拍照 → 吴山广场新丰小吃晚餐 → 吴山俯瞰老城区夜景

5日深度游

第1天

Day 4：良渚古城遗址 + 拱宸桥运河街区

地铁2号线到良渚站 → 良渚古城遗址公园（60元，世界遗产）→ 良渚博物院（免费）→ 良渚文化村午餐 → 拱宸桥运河历史街区 → 中国京杭大运河博物馆（免费）→ 沿河晚餐

第2天

Day 5：云栖竹径 + 梅家坞茶村 + 虎跑公园 + 钱塘江大桥

公交到云栖竹径（8元，竹林清幽）→ 梅家坞茶村（农家菜午餐）→ 虎跑公园（15元）→ 钱塘江大桥（中国第一座公铁两用桥）→ 闻涛路看江景 + 六和塔（20元）

06常见问题

西湖玩几天合适？

建议预留一整天给西湖。环湖一圈约10公里，如果只走核心段（断桥到雷峰塔）半天也够。想深度体验的话，一天骑行+步行+游船的组合最舒服。上午从断桥出发走白堤到孤山，中午在楼外楼吃饭，下午沿苏堤走到花港观鱼，傍晚坐船到三潭印月看日落。如果只有半天时间，建议走北线（断桥-白堤-孤山-苏堤北段），这一段浓缩了西湖精华。

西湖要门票吗？

西湖景区本身免费开放，24小时可进入。部分小景点需要单独购票：雷峰塔40元、岳王庙25元、三潭印月船票+登岛55元、灵隐寺30元（不含飞来峰门票45元）。环湖骑行租共享单车约1.5元/30分钟，景区电瓶车环湖一圈40元/人。

灵隐寺怎么去？交通方便吗？

灵隐寺位于西湖西北侧，交通很方便。公交7路（城站火车站始发，经龙翔桥、岳庙）直达灵隐站，票价2元，全程约40分钟。公交407路（地铁武林广场站始发）也可到达。打车从西湖湖滨出发约15-20分钟，费用20-30元。地铁3号线黄龙体育中心站下车后换乘公交7路或打车约10分钟即到。建议早上7:30前到达，避开旅行团高峰。

杭州住宿推荐哪个区？

推荐三个区域：首选湖滨/龙翔桥一带（地铁1号线沿线），步行到西湖5分钟，连锁酒店300-500元/晚。第二选择是武林广场/凤起路周边（地铁1号、2号线交汇），去西湖地铁1站即达，250-400元/晚。第三选择是青芝坞/玉泉一带，靠近浙大玉泉校区和灵隐寺，民宿200-350元/晚，适合喜欢安静和自然氛围的旅客。

杭州有什么必吃的美食？

杭州必吃五道：西湖醋鱼（楼外楼草鱼版78元/条）、东坡肉（知味观18元/客，肥而不腻配米饭绝佳）、片儿川（菊英面馆18-25元/碗，雪菜+笋片+肉丝）、龙井虾仁（绿茶餐厅68-88元/份，虾仁嫩滑带茶香）、葱包桧（街头小吃3-5元/个，南宋流传至今）。早点去新丰小吃吃小笼包（10元/笼）也很本地。杭州菜整体偏清淡偏甜，口味重的可以加本地辣酱。

龙井村怎么玩？适合徒步吗？

龙井村非常适合徒步。坐公交27路或87路到龙井茶室站，从龙井村穿村而过走九溪十八涧步道，全程约5公里/1.5-2小时，全程缓坡下行不费力。沿途是茶园、溪涧和枫树，茶农家喝茶免费。走完后从九溪公交站坐车回市区。建议穿防滑鞋，避开雨后石板路湿滑的情况。

杭州什么季节去最美？

春季（3-5月）和秋季（9-11月）是最佳季节。春天苏堤春晓、龙井明前茶采摘，气温15-25°C。秋天满城桂花香、九溪红叶、钱塘江大潮（农历八月十八前后），气温20°C左右。夏季（6-8月）热且潮湿（35-38°C），但荷花盛开是曲院风荷最美的时段。冬季（12-2月）游客最少，断桥残雪是经典画面，住宿价格最低。

杭州周边有什么值得去的地方？

杭州周边一日游选择很多。乌镇（高铁到桐乡站转公交，约1.5小时，西栅150元看水乡夜景）、千岛湖（大巴2小时，门票+船票185元起，夏季避暑）、绍兴（高铁19分钟即到，鲁迅故里免费，黄酒乌篷船）、莫干山（高铁到德清站转车，门票80元，民宿和避暑胜地）、安吉（大巴1.5小时，大竹海门票58元）。时间紧的话优先乌镇或绍兴，交通最方便。
